# Qt_OpenCV_Weld_Tracking
焊缝跟踪

1.利用树莓派和工业相机搭建主动激光焊缝提取系统，通过Steger算法以及对Shi-Tomasi角点检测算法进行精度优化，做到了亚像素精度级别的焊缝识别，得到了坡口处的位置信息，发送给下位机。

2.实现了Qt和OpenCV库的联合使用，编写GUI界面完成焊缝跟踪界面。
